Hotel Management System

A Python and SQLite-based Hotel Management System with a user-friendly GUI built using Tkinter. This system allows hotel staff to manage customers, rooms, reservations, payments, and staff efficiently through a dashboard interface.

Features
Authentication

Login: Secure login system for staff members.

Signup: Create new staff accounts with username and password validation.

Dashboard

Sidebar Navigation: Easily navigate between different sections.

Dashboard Overview: Quick stats and insights about hotel operations.

Management Modules

Customers

Add, update, and delete customer details.

View customer information.

Rooms

Manage room availability, types, and pricing.

Add new rooms and update existing rooms.

Reservations

Book rooms for customers.

View current and past reservations.

Update or cancel reservations.

Payment

Track customer payments and invoices.

Record payment details.

Staff

Manage staff members and their roles.

Add, update, or remove staff accounts.

Technologies Used

Python – Main programming language.

Tkinter – GUI library for desktop interface.

SQLite – Lightweight relational database for storing hotel data.

hashlib / bcrypt (optional) – For password hashing and secure authentication.
